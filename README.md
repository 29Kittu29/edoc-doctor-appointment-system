# 🩺 Doctor Appointment Booking System (PHP + MySQL)

A web‑based **Doctor Appointment Booking System** built using **PHP, MySQL, HTML, and CSS**.  
This project enables patients to book appointments online, doctors to manage their schedules, and administrators to oversee the system.  

Developed and customized by **Kritika Ranjan** as part of a college project and portfolio showcase.

---

## 🚀 Features

### 👨‍💼 Admin
- Add, edit, and delete doctors  
- Schedule or remove doctor sessions  
- View patient details and bookings  

### 👩‍⚕️ Doctor
- View appointments and scheduled sessions  
- Manage account settings  
- Access patient details  

### 🧑‍🤝‍🧑 Patient
- Create an account and book appointments online  
- View past bookings  
- Edit or delete account
